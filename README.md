# perfectPitch

Minimal Vite + TypeScript setup.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## GitHub Pages

The `base` is set to `/perfectPitch/` in `vite.config.ts` for repo-subpath deployment.

## SoundFont

Reference tones use the FluidR3_GM soundfont via midi-js-soundfonts (CC BY 3.0).
Source: https://github.com/gleitz/midi-js-soundfonts
