# 仕様書（perfectPitch）

## UI構成（1画面）

### ヘッダー
- ラベル: `MVP`
- タイトル: `perfectPitch`
- 説明: 「開始」ボタンでマイクを許可し、音名/Hz/centsを推定

### メイン操作パネル
- **ステータス表示**: READY/WAIT/LIVE/ERROR（表示文言も併記）
- **操作ボタン**:
  - 開始/停止
  - 再試行（エラー時）
  - 基準音を鳴らす
  - 1オク下を鳴らす
- **録音**:
  - 録音開始/停止
  - 再生（録音停止後に有効）
- **現在値表示**:
  - CURRENT NOTE（音名）
  - TARGET（選択）
- **メトリクス**:
  - FREQ
  - CENTS (TARGET)
  - CONF
- **入力チューニング**:
  - マイク感度（RMSしきい値とconfidenceしきい値を連動調整）
  - 音程取得速度（平滑化と安定化フレーム数を調整）
- **ゲージ**:
  - -50¢ / 0 / +50¢ のスケール
  - 信号なし時は NO SIGNAL
- **デバッグ**:
  - 波形（先頭48サンプル）をテキスト表示

### 録音の振り返り
- メロディ（メロディ再生/再生速度）
- 簡易統計
- スコア
- 今日の課題
- 時系列（cents）: グラフ表示
- 時系列リスト（テキスト）

### チェックポイント
- 動作確認用の箇条書き

## 操作フロー（概要）
1. 「開始」でマイク権限を取得し、ピッチ解析を開始
2. 解析結果をリアルタイム表示（音名/Hz/cents/conf）
3. 必要に応じてターゲット音を変更
4. 録音開始→停止でレビュー生成
5. レビュー後、メロディ再生が可能
