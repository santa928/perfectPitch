# 要件定義書（perfectPitch）

## 目的
- ブラウザ上で **マイク入力から音名/Hz/cents を推定** し、ボーカル練習に使えるフィードバックを提供する。

## 対象ユーザー
- ピッチを安定させたい歌唱練習者
- iPhone Safari / Android 等モバイル利用者を含む

## 提供価値（MVP）
- 現在の音程をリアルタイム表示（音名・周波数・ターゲットとの差）
- ターゲット音を指定してズレを可視化
- 録音して振り返り（簡易統計・スコア・課題提示・時系列表示）
- 録音フレーズを簡易メロディとして再生（速度変更）

## 画面要件
- **単一画面**で完結する（ヘッダー、メイン操作パネル、録音の振り返り、チェックポイント）。

## 成果物（UI書き出し）
